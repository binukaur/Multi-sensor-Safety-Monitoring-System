# Multi-sensor-Safety-Monitoring-System
A wearable, on-device safety system that detects abnormal struggle-like motion using multimodal sensing and triggers evidence capture autonomously.


## Hardware Used
- Arduino Nicla Vision
- Computer (for displaying whether the person is in danger or not)

## Demonstration Videos
- Normal Activity
- Danger detection by voice
- Struggle Detection

## Privacy & Ethics
- No continuous recording
- Camera activates only during alerts
- No cloud storage
- All processing is local

## Future Work
- Audio detection integration
- Smartphone / gateway support
- SOS notifications
- Multi-wearable fusion
